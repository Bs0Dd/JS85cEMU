Options panel


    The options panel is a tool for configuring and managing the emulator, allowing you to modify basic parameters that affect the behavior of the virtual encryptor.

    The panel consists of 4 sections with parameters grouped according to common characteristics.


Speed configuration section


    This section allows you to switch the speed modes of the encryptor and change the speed of these modes.
    The first line ("Speed") shows the current state of the processor. It can be normal ("Normal") or accelerated ("Turbo"). The 8x clock divider can also be enabled ("(8x slowed by freq. divider)") or the processor clock can be stopped ("(clock stopped)").
    The "Standard" and "Turbo" lines are used to change the speed of the normal and turbo modes, respectively. The default speed for normal mode is 25000 operations per second and 120000 operations per second for turbo. These values are close to real encryptors (the speed in them floats from device to device, which is a consequence of the imperfection (scatter) of the resistors used in the RC clock circuit, no quartz resonators), therefore they are considered nominal. The "Reset" button for each mode will always set its nominal value. To change (in the range from 100 to 1000000 operations per second), just enter the desired value in the input field and click "Set".
    The "Change speed mode" button allows you to change the mode using hot swap (without rebooting the emulator). It is recommended to change the mode only when the device's clock generator is stopped. If you make a change while scanning the keyboard or outputting text, the mode may not switch. The "Restart in turbo mode" button restarts the emulator with the turbo mode enabled.

    Unfortunately, the existing firmware of the MK85C, when accessing the CPUC register, constantly resets the turbo mode bit, so it is not possible to enable it.


Control section


    The section is used to control the operation of the emulator and switch general functions. The "Pause" and "Restart" buttons, as the name suggests, are used to pause and restart the emulator. The "Vibro (keys)" parameter turns on or off the vibration of the device (if it is capable of vibrating, which usually applies to portable devices) when you press the virtual keyboard of the encryptor. On devices without vibration capabilities, the item does not perform any function.


Additional functions section


    The section contains buttons for opening utilities and information windows, as well as debugging parameters.
    The "Keyboard layout" button opens a window with a computer keyboard layout diagram adapted for working with a virtual encryptor. The "Xchanger" button opens the Xchanger utility panel. The "Debugger" button opens the Debugger panel. The "Help" button opens a window with this help. And the "About" button opens a window with brief information about the program and a list of changes (Changelog).
    The "Meaningful display" parameter changes the text segments of the display (the MK85C uses a standard display, but the segments have different purposes) so that they correspond to the assigned indication (RUN -> РШФ (DEC), WRT -> ШИФ (ENC), etc.). The "Force turbo" parameter is used to force the turbo mode usage, since the real device, when turned on with the turbo mode, is not actually in it permanently, but turns it off when outputting text on the display and scanning the keyboard. Parameter "Ignore frq. div." forcibly disables the processor clock divider (8 times), also used when outputting text on the display and scanning the keyboard. The "Disable soft poff" parameter disables support for the software switch present in the processor, which allows the firmware to turn off the device. The "Show debug messages" parameter allows debug messages to be shown in the console (via the console.log function).


Data loading and saving section


    In this section you can load or save RAM and ROM of the virtual encryptor. The "RAM/ROM file" fields display the names of loaded files for RAM and ROM memory. The inscription "internal" is shown if the built-in (factory) ROM is used or if the RAM image is created in the emulator. To load RAM or ROM from a binary file, just select the file (using the button or via Drag'n'Drop) and click the "Load RAM/ROM from file" button. In this case, for RAM the file must be equal of 2KB, for ROM - multiple of 8KB. ROM file cannot exceed 32KB. If the conditions are not met, the images will be automatically expanded to multiples of the corresponding numbers, or trimmed to 2KB / 32KB.
    To reset the ROM image to the factory one, just press the "Reset ROM" button. RAM can be reset by pressing "Clear RAM" button. The "Fast init" button performs a fast initialization of RAM, which allows you to skip the device test performed during normal initialization.
    RAM can be saved to disk as a binary file. This is done with the "Save RAM" button, and the file "mk85c_ram.bin" will be downloaded by the browser.

    The "Switch to model Б/C" button is used to switch the emulator between MK85C and MK85Б (banking) emulation modes. The MK85Б (very rare even in comparison with the MK85C) has other keys in the number block (on the case it is still designated as MK85C), is equipped with 4KB of RAM and uses its own firmware to encrypt bank advice. It was used by employees of the Central Bank of the Russian Federation, but unlike the MK85C, it is not commercial, and, according to some information, is intended only for official use. For this reason, at this moment I can't add ROM to the emulator, so mode Б only changes the appearance of the keys and expands the RAM, the firmware (not available publicly) must be loaded manually.
