"Xchanger" utility


    The encrypter is designed to encrypt and decrypt data. However, MK85C does not have any available ports for data exchange, so text can only be entered using the keyboard, and the result must be rewritten from the display.

    The Xchanger utility allows you to automate the input and extraction of data from the encrypter, and also has additional functionality.

    To enter data into the encrypter, you must press the "Send to MK" button. By default, data is transferred (similar to real use) via emulated keystrokes. This is a reliable, but very slow method, ill-suited for entering large data. The "directly" parameter enables the mode of entering data directly into RAM, while the utility itself receives the necessary addresses and parameters for correct input, and then modifies the state of the virtual encrypter. This mode also allows entering characters that are not available for input via the keyboard, but present in the encrypter code page (modified KOI-7). In case direct input to RAM is impossible, data will be transferred via keystroke emulation. The utility distinguishes what type of data (symbols or only digits) the encrypter expects, automatically filtering the input (symbols missing in the MK85C encoding are also ignored).

    The "Get result" button extracts the encryption or decryption result from the encrypter. The utility automatically determines the type of data (symbols or digits only). If numeric data is read, it is automatically split into blocks (groups), the length of which is specified during configuration of the encrypter.

    The "Get DKL" button allows you to extract the current long-term key (DKL / LTK) from the device, which is 110 digits (100 key digits and 10 checksum digits), which are always split into blocks of 5 digits.

    The "New DKL" button generates a new long-term key with a correct checksum. ATTENTION: the Math.random() function is used to generate the key.

    The "Copy" and "Clear" buttons are used to copy and clear the data in the input field, respectively.

    The "Close" button closes the utility window.
