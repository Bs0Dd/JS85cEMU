Frequently asked questions (FAQ)

    Q: How to use the encryptor? Can not understand anything!
    A: Read the unofficial guide, which contains all the information on working with the encryptor. The manual is available in HTML and PDF (Russian only) versions.



    Q: Why does the encryptor show a question mark?
    A: You probably created a new RAM image, but it was not initialized (initial settings writing). For manual initialization, just press the "начальная установка" (initial setup) button, after which the encryptor will be tested and the RAM will be initialized. You can skip the test using the "Fast init" button on the options panel. You can read about further configuration in the manual.



    Q: Why does the encryptor display the string "** МК-85С **" and does not perform any actions?
    A: The encryptor is initialized, but not yet configured. To start the configuration, you need to enter (not shown) the code word "АЗИМУТ" (azimuth). You can read about further configuration in the manual.



    Q: Why does the encryptor display the string " А З И М У Т" and does not perform any actions?
    A: The encryptor is locked and waiting for a password to be entered. After 25 incorrectly entered characters, for security purposes, the encryptor will ERASE all information in RAM. Read more here.



    Q: How can I get a DKL (LTK) for the encryptor?
    A: A long-term key can be generated directly in the emulator using the "Xchanger" utility (the "New DKL" button). Alternatively, DKLKM85C program or utility for Python by kaseiiro is available.



    Q: What is "model Б" and why does pressing this button only change the RAM value and the appearance of some keys?
    A: The "Switch to model Б/С" button is used to switch the emulator between MK85C and MK85Б (banking) emulation modes. The MK85Б (very rare even in comparison with the MK85C) has other keys in the number block (on the case it is still designated as MK85C), is equipped with 4KB of RAM and uses its own firmware to encrypt bank advice. It was used by employees of the Central Bank of the Russian Federation, but unlike the MK85C, it is not commercial, and, according to some information, is intended only for official use. For this reason, at this moment I can't add ROM to the emulator, so mode Б only changes the appearance of the keys and expands the RAM, the firmware (not available publicly) must be loaded manually.
