Утилита "Xchanger"


    Шифратор предназначен для шифрования и расшифровки данных. Однако МК85С не имеет каких-либо доступных портов для обмена данными, поэтому текст можно ввести только с помощью клавиатуры, а результат нужно переписывать с дисплея.

    Утилита Xchanger позволяет автоматизировать ввод и извлечение данных из шифратора, а также имеет дополнительный функционал.

    Для ввода данных в шифратор необходимо нажать кнопку "Send to MK". По умолчанию, данные передаются (подобно реальному использованию) через эмуляцию нажатий клавиш. Это надежный, но очень медленный способ, плохо подходящий для ввода больших данных. Параметр "directly" включает режим ввода данных напрямую в ОЗУ, при этом утилита сама получает нужные адреса и параметры для корректного ввода, а затем модифицирует состояние виртуального шифратора. Этот режим также позволяет вводить символы, недоступные для ввода через клавиатуру, но присутствующие в кодовой странице шифратора (модифицированная КОИ-7). В случае, если прямой ввод в ОЗУ невозможен, данные будут переданы через эмуляцию нажатий клавиш. Утилита различает, какой тип данных (символы или только цифры) ожидает шифратор, автоматически фильтруя ввод (также игнорируются символы, отсутствующие в кодировке МК85С).

    Кнопка "Get result" извлекает из шифратора результат шифровки или расшифровки. Утилита автоматически определяет тип данных (символы или только цифры). Если считываются числовые данные, они автоматически разбиваются на блоки (группы), длина которых задается при настройке шифратора.

    Кнопка "Get DKL" позволяет извлечь из устройства текущий долговременный ключ (ДКЛ), представляющий из себя 110 цифр (100 цифр ключа и 10 цифр контрольной суммы), которые всегда разбиты на блоки по 5 цифр.

    Кнопка "New DKL" генерирует новый долговременный ключ с корректной контрольной суммой. ВНИМАНИЕ: для генерации ключа используется функция Math.random().

    Кнопки "Copy" и "Clear" служат, соответственно, для копирования и очистки находящихся в поле ввода данных.

    Кнопка "Close" закрывает окно утилиты.
