    JS85cEMU is an emulator of the Soviet/Russian encryptor known as Elektronika MK85C or "Ancript". This device differs from the Elektronika MK85 microcomputer only in its modified design and special firmware.

    The task of the emulator is to emulate the device as accurately as possible, taking into account its technical features. This emulator introduces software switch and Turbo mode emulation for the first time. It also contains a PDP-11 debugger (microcomputer processor architecture) and "Xchanger" utility for more convenient interaction with MK85C.

    The help describes the features of working with the emulator and its subsystems. However, it also includes a manual to working with the encrypter, because it is a very rare device that has been used by very few people. The official documentation is considered lost, so enthusiasts have created an unofficial manual describing the intricacies of the work. You can also download PDF version (Russian).
